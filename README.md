# Dougbro55

A host portal and AI assistant for real estate agents.

## Tech stack

- [Next.js 15](https://nextjs.org) (App Router)
- React 19 + TypeScript

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the welcome page.

## Project structure

```
app/
  layout.tsx        # Root layout + metadata
  page.tsx          # Welcome / landing page
  globals.css       # Global styles + theme tokens
  page.module.css   # Landing page styles
```
